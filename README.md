## Описание проекта

Проект представляет собой интеллектуального помощника (чат-бота) для автоматизации процесса записи клиентов на консультации к нутрициологу. Бот способен вести осмысленный диалог, отвечать на вопросы по базе знаний и управлять расписанием встреч через интеграцию с Google Календарем.

Цель проекта — повысить эффективность взаимодействия между клиентами и нутрициологом, минимизировать рутинные задачи и обеспечить удобный пользовательский опыт.

---

## Функционал

1. **Осмысленное ведение диалога**  
   - Чат-бот использует современные технологии обработки естественного языка для поддержания контекста разговора.
   - Умение задавать уточняющие вопросы для точного понимания запроса клиента.

2. **Ответы на вопросы по базе знаний**  
   - База знаний содержит информацию о принципах работы нутрициолога, услугах, ценах и часто задаваемых вопросах.
   - Возможность обучать бота новым данным для актуализации информации.

3. **Запись на встречу**  
   - Клиент может выбрать удобное время для консультации.
   - Автоматическое добавление встречи в Google Календарь нутрициолога.

---

## Используемые сервисы

### 1. **Qwen**
   - **Назначение**: Создание системного промпта и формирование базы знаний.
   - **Особенности**: Qwen предоставляет мощные инструменты для генерации текстов и управления базой знаний, что позволяет создать качественную основу для чат-бота.

### 2. **Савви (Suvvy)**  
   - **Сайт**: [https://suvvy.ai/](https://suvvy.ai/)  
   - **Назначение**: Платформа для создания ИИ-ассистента.  
   - **Особенности**:  
     - Савви работает на базе надёжных LLM (больших языковых моделей).  
     - Обеспечивает ответы на 100% обращений с точностью ответов от 95%.  
     - Интеграция с CRM, мессенджерами и маркетплейсами для расширения функционала.  

---

## Архитектура проекта

1. **Взаимодействие с пользователем**  
   - Чат-бот доступен через популярные мессенджеры (например, Telegram, WhatsApp) или веб-интерфейс.

2. **Обработка запросов**  
   - Входящие сообщения обрабатываются через платформу Савви, которая использует предварительно настроенный системный промпт и базу знаний.

3. **Интеграция с Google Календарем**  
   - При записи на встречу данные передаются в Google Календарь через API для автоматического создания события.

4. **Административная панель**  
   - Интерфейс для управления базой знаний, просмотра статистики и настройки расписания.
